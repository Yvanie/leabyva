# Leabyva Digital Works

Bienvenue sur le dépôt du site vitrine de **Leabyva Digital Works**. Ce projet est une application web (Single Page Application - SPA) de qualité premium.

## 🚀 Fonctionnalités
- **Design Premium** : Charte graphique moderne avec un mode sombre par défaut, effets de verre (glassmorphism) et animations cinématiques.
- **Single Page Application (SPA)** : Navigation fluide sans rechargement de page.
- **Routage Hash** : Les URL sont conservées via le hash (ex: `#blog`, `#port`), permettant le partage de liens et le rafraîchissement.
- **Multi-langue** : Support complet du Français (FR) et de l'Anglais (EN).
- **Thème personnalisable** : Bascule entre le mode sombre et le mode clair.

## 📂 Structure du Projet
- `app/` - Contient l'application divisée en composants.
  - `Views/` - Les différentes pages (Accueil, Services, À propos, Portfolio, Blog, Contact, Layout principal) et les partiels (Header).
- `Public/` - Contient les fichiers statiques.
  - `css/style.css` - Le cœur du design.
  - `js/script.js` - Logique front-end (Routage, Thème, Langue, Formulaires).
  - Images et ressources visuelles.
- Fichiers à la racine - Composants système et maquettes HTML originales.

## 🛠️ Installation & Lancement en local

1. Clonez ce dépôt sur votre machine locale.
2. Assurez-vous d'avoir PHP installé (version >= 7.4).
3. Ouvrez un terminal à la racine du projet et lancez le serveur PHP intégré :
   ```bash
   php -S localhost:7400
   ```
4. Ouvrez votre navigateur et accédez à [http://localhost:7400](http://localhost:7400).
